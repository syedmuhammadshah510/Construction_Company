## ✨ Features

- **Animated Loader** — Progress bar with pulsing icon, auto-dismisses after load
- **Sticky Navbar** — Transparent → dark on scroll, active link highlight, mobile hamburger drawer
- **Hero Slider** — Auto-playing 3-slide background with dot navigation and CTA buttons
- **Marquee Strip** — Scrolling service ticker in brand orange
- **About Section** — Layered image layout with 25+ years badge and feature grid
- **Services Grid** — 6 service cards with Unsplash images and hover animations
- **Stats Counter** — Animated number counters triggered on scroll (500+ projects, 150+ team, etc.)
- **Project Portfolio** — 6 project cards with category filter (All / Commercial / Residential / Infrastructure)
- **Why Choose Us** — Dark section with numbered feature highlights
- **Team Section** — 4 leadership cards with social media hover overlay
- **Testimonials Carousel** — Sliding cards with prev/next controls
- **CTA Banner** — Full-bleed parallax-style section
- **Contact Section** — Info panel + validated form with service dropdown
- **Footer** — 4-column layout with newsletter signup and ISO badges
- **Scroll-to-Top Button** — Appears after scrolling 400px
- **Scroll Reveal Animations** — Elements animate in as you scroll

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `> 1100px` | Full desktop layout |
| `≤ 1100px` | Services 2-column, footer 2-column |
| `≤ 900px` | Single column about/why-us, 2-col team & projects |
| `≤ 768px` | Mobile nav drawer, stacked sections |
| `≤ 480px` | Single column everything, stacked hero buttons |

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/syedmuhammadshah510/Construction_Company.git
   ```

2. **Navigate to the folder**
   ```bash
   cd Construction_Company
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in any modern browser
   start index.html        # Windows
   open index.html         # macOS
   xdg-open index.html     # Linux
   ```

No installation, no build process, no package manager needed.
